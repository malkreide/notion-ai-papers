# Beitragen zum Notion AI Papers Skill

Vielen Dank für dein Interesse, zu diesem Projekt beizutragen! 

## Wie kann ich beitragen?

### Fehler melden

Wenn du einen Fehler findest:

1. Überprüfe, ob der Fehler bereits als Issue gemeldet wurde
2. Falls nicht, erstelle ein neues Issue mit:
   - Beschreibung des Fehlers
   - Schritte zur Reproduktion
   - Erwartetes vs. tatsächliches Verhalten
   - Claude-Version und Kontext (falls relevant)

### Verbesserungen vorschlagen

Für Feature-Requests oder Verbesserungen:

1. Erstelle ein Issue mit dem Label `enhancement`
2. Beschreibe das gewünschte Feature detailliert
3. Erkläre den Nutzen und Anwendungsfall
4. Diskutiere die Umsetzung (optional)

### Code beitragen

1. **Fork** das Repository
2. Erstelle einen **Feature Branch** (`git checkout -b feature/AmazingFeature`)
3. **Committe** deine Änderungen (`git commit -m 'Add some AmazingFeature'`)
4. **Pushe** zum Branch (`git push origin feature/AmazingFeature`)
5. Öffne einen **Pull Request**

### Pull Request Richtlinien

- Beschreibe deine Änderungen klar und präzise
- Referenziere relevante Issues
- Halte Änderungen fokussiert (ein Feature/Fix pro PR)
- Teste deine Änderungen gründlich
- Aktualisiere die Dokumentation bei Bedarf
- Nutze schweizerische Rechtschreibung für deutschsprachige Texte

## Anpassung für andere Kontexte

Dieser Skill ist speziell auf das Schulamt der Stadt Zürich zugeschnitten. Wenn du ihn für deinen eigenen Kontext anpassen möchtest:

1. **Fork** das Repository
2. Passe folgende Dateien an:
   - `references/user-context.md` - Dein Rollenprofil und Perspektiven
   - `SKILL.md` - Notion-Datenbank-ID und Workflow
   - `references/notion-fields.md` - Deine Felddefinitionen

3. Dokumentiere deine Änderungen im README
4. Teile deine Anpassung gerne als Inspiration für andere!

## Verhaltenskodex

- Sei respektvoll und konstruktiv
- Fokussiere auf den Inhalt, nicht auf Personen
- Hilf anderen, zu lernen und sich zu verbessern
- Respektiere unterschiedliche Meinungen und Ansätze

## Fragen?

Bei Fragen zum Projekt oder zum Beitragen:

1. Suche in den bestehenden Issues
2. Erstelle ein neues Issue mit dem Label `question`
3. Kontaktiere die Maintainer
