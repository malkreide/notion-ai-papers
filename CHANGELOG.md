## [1.1.0] - 2026-02-18

### Added
- Bilingual documentation (English + German)
- `README.md` (English), `README.de.md` (German)
- `CONTRIBUTING.md` (English), `CONTRIBUTING.de.md` (German)
- `INSTALLATION.md` (English), `INSTALLATION.de.md` (German)
- `CHANGELOG.md` (English), `CHANGELOG.de.md` (German)
- Language switcher links in all documentation files

### Changed
- Previous German `README.md` moved to `README.de.md`
- Previous German `CONTRIBUTING.md` moved to `CONTRIBUTING.de.md`
- Previous German `INSTALLATION.md` moved to `INSTALLATION.de.md`
- Previous German `CHANGELOG.md` moved to `CHANGELOG.de.md`
- Updated file structure documentation in both READMEs

### Unchanged
- `SKILL.md` remains in German (operational skill for German-speaking contexts)
- `references/` files remain in German
- `LICENSE` remains in English

## [1.0.0] - 2026-02-15

### Added
- Initial release of the Notion AI Papers Skill
- Systematic paper analysis workflow (5-step process)
- Multi-source paper retrieval (arXiv, DOI, ScienceDirect, Google Drive)
- Notion database integration with structured property mapping
- Automatic reporting with relevance assessments
- Focus area specializations (Education, Special Education, Public Administration, AI Governance)
- Reference files for user context and Notion field definitions
- Optional PDF analyzer helper script
- Comprehensive documentation with installation guides
- MIT License

### Features
- Automatic metadata extraction from research papers
- Classification system (Topics, UseCase, Target Audience)
- Quality criteria for GL-level summaries
- Error handling for inaccessible papers and API failures
- Swiss German spelling conventions

### Documentation
- Full SKILL.md with detailed workflow
- README with installation and usage instructions
- Contributing guidelines
- Installation guide with 4 integration options
- Changelog
